# maze maker

## 迷宫生成器

- 项目框架 [Vue.js](https://cn.vuejs.org)
- 项目UI [Arco Design](https://arco.design/vue)
- 状态管理库 [Pinia](https://pinia.vuejs.org/zh)

# 安装依赖

`npm install`

# 编译项目

`npm run build`

# 运行编译前的项目

`npm run dev`

# 运行编译后的项目

`npm run preview`
